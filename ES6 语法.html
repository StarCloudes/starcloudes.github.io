ES6 Transpilation

In this lesson, you will learn about two important tools for addressing browser compatibility issues.

caniuse.com ‚Äî A website that provides data on web browser compatibility for HTML, CSS, and JavaScript features. You will learn how to use it to look up ES6 feature support.

Babel ‚Äî A Javascript library that you can use to convert new, unsupported JavaScript (ES6), into an older version (ES5) that is recognized by most modern browsers.

  1. caniuse.com

While most new browser versions support the majority of the ES6 library, there are still a couple sources of compatibility issues:

  • Some users have not updated to the latest, ES6 supported web browser version.
  • A few ES6 features, like modules, are still not supported by most web browsers.

Because companies add support for ES6 features gradually, it's important for you to know how to look up browser support on a feature-by-feature basis. The website caniuse.com is the best resource for finding browser compatibility information.

In caniuse, you can enter an ES6 feature, like let, and see the percentage of browsers that recognize it. You can also see when each major web browser (Chrome, Safari, Edge, etc.) added support for the keyword.

  1. Why ES6?

  • A similarity to other programming languages ‚Äî JavaScript ES6 is syntactically more similar to other object-oriented programming languages. This leads to less friction when experienced, non-JavaScript developers want to learn JavaScript.
  • Readability and economy of code ‚Äî The new syntax is often easier to understand (more readable) and requires fewer characters to create the same functionality (economy of code).

  • Addresses sources of ES5 bugs ‚Äî Some ES5 syntax led to common bugs. With ES6, Ecma introduced syntax that mitigates some of the most common pitfalls.
